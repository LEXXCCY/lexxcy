Perintah Dasar Debian 11

Pelajari perintah-perintah dasar untuk administrasi sistem Debian 11

Manajemen File dan Direktori

  • ls

    Menampilkan daftar file dan direktori

    $ ls -la
  • cd

    Berpindah antar direktori

    $ cd /home/user
  • mkdir

    Membuat direktori baru

    $ mkdir folder_baru
  • rm

    Menghapus file atau direktori

    $ rm -rf folder_lama

Manajemen Sistem

  • apt update

    Memperbarui daftar paket

    $ sudo apt update
  • apt install

    Menginstal paket baru

    $ sudo apt install nama_paket
  • systemctl

    Mengontrol layanan sistem

    $ sudo systemctl start apache2
  • reboot

    Memulai ulang sistem

    $ sudo reboot

Manajemen Jaringan

  • ip addr

    Menampilkan konfigurasi IP

    $ ip addr show
  • ping

    Menguji konektivitas jaringan

    $ ping -c 4 google.com
  • netstat

    Menampilkan koneksi jaringan

    $ netstat -tupln
  • ssh

    Koneksi remote secure shell

    $ ssh user@host

Manajemen Pengguna

  • useradd

    Membuat pengguna baru

    $ sudo useradd username
  • passwd

    Mengubah password pengguna

    $ sudo passwd username
  • usermod

    Memodifikasi akun pengguna

    $ sudo usermod -aG sudo username
  • who

    Menampilkan pengguna yang login

    $ who
