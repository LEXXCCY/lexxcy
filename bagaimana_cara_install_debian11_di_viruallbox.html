Tutorial Instalasi Debian 11 di VirtualBox

Panduan lengkap cara menginstal Debian 11 (Bullseye) di Oracle VirtualBox

Durasi: 30-45 menit
Level: Pemula
10 Langkah

Persyaratan

  • VirtualBox sudah terinstal di komputer
  • File ISO Debian 11 (minimal 4GB ruang disk)
  • RAM minimal 2GB
  • Koneksi internet (opsional)
1

Membuat Virtual Machine Baru

Buka VirtualBox dan klik tombol "New" untuk membuat virtual machine baru.

Tips: Beri nama yang mudah diingat untuk VM Anda, misalnya "Debian11"
Tampilan menu New Virtual Machine
2

Konfigurasi Memory (RAM)

Alokasikan minimal 2GB (2048MB) RAM untuk performa yang optimal.

Tips: Jangan mengalokasikan lebih dari 50% RAM fisik komputer Anda
3

Membuat Virtual Hard Disk

Pilih "Create a virtual hard disk now" dan tentukan ukuran disk minimal 20GB.

Pilih format VDI (VirtualBox Disk Image) dan pilih "Dynamically allocated" untuk menghemat ruang disk.

4

Memulai Instalasi Debian

Start VM dan pilih file ISO Debian 11 yang sudah didownload.

Tips: Pastikan checksum ISO sesuai dengan yang ada di website Debian
5

Pemilihan Bahasa dan Lokasi

Pilih bahasa dan lokasi sesuai preferensi Anda.

  • Bahasa: English/Indonesia
  • Location: Other → Asia → Indonesia
  • Keymap: American English
6

Konfigurasi Network

Pilih interface network dan konfigurasikan hostname.

Tips: Gunakan NAT untuk koneksi internet yang mudah
7

Partisi Disk

Pilih metode partisi yang diinginkan:

  • Guided - use entire disk (untuk pemula)
  • Manual (untuk partisi custom)
Tips: Untuk pemula, gunakan opsi Guided partitioning
8

Pengaturan User

Buat user dan password root:

  • Set root password
  • Buat user baru
  • Set password user
9

Pemilihan Software

Pilih paket software yang akan diinstal:

  • Debian desktop environment
  • GNOME (default)
  • Web server
  • SSH server
  • Standard system utilities
10

Finalisasi Instalasi

Tunggu proses instalasi selesai dan sistem akan restart.

Tips: Setelah restart, install VirtualBox Guest Additions untuk performa lebih baik
