Cara Top Up Diamond Mobile Legends

1
Temukan User ID dan Server
1. Buka Mobile Legends
2. Klik foto profil di pojok kiri atas
3. User ID dan Server terletak di bawah nama profil
4. Format: 12345678 (1234)
2
Pilih Geme
1. Pilih geme yang Anda inginkan di top up kan
3
Pilih Jumlah Diamond
1. Pilih paket diamond yang Anda inginkan
2. Perhatikan jumlah diamond dan harga
3. Pastikan saldo mencukupi untuk pembelian
4
Pilih Metode Pembayaran
1. Pilih metode pembayaran (DANA/OVO/GoPay)
2. Pastikan akun pembayaran Anda aktif
3. Siapkan saldo yang cukup
5
Konfirmasi dan Bayar
1. Klik tombol "Beli Sekarang"
2. Anda akan diarahkan ke WhatsApp
3. Ikuti instruksi pembayaran
4. Diamond akan masuk dalam 1-5 menit
