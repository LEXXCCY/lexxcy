Tutorial Install SSH Server di Debian 11

SSH (Secure Shell) adalah protokol jaringan yang memungkinkan pertukaran data yang aman antara dua komputer. Tutorial ini akan memandu Anda langkah demi langkah dalam menginstal dan mengkonfigurasi SSH Server di Debian 11.

Persyaratan

  • Debian 11 terinstal di VirtualBox
  • Koneksi internet aktif
  • Akses root atau sudo
  • Basic understanding of terminal commands

1 Update Sistem

Pertama, update sistem Debian Anda dengan menjalankan perintah berikut:

sudo apt update && sudo apt upgrade -y

2 Install SSH Server

Install OpenSSH server dengan perintah:

sudo apt install openssh-server -y

Setelah instalasi, SSH server akan otomatis dijalankan sebagai service.

3 Verifikasi Status SSH

Periksa status SSH service dengan perintah:

sudo systemctl status ssh

Anda seharusnya melihat status "active (running)"

4 Konfigurasi Firewall

Jika Anda menggunakan UFW firewall, izinkan koneksi SSH:

sudo ufw allow ssh

Pastikan untuk mengizinkan port SSH (22) sebelum mengaktifkan firewall untuk menghindari terkunci dari sistem.

5 Konfigurasi SSH

Edit file konfigurasi SSH:

sudo nano /etc/ssh/sshd_config

Beberapa pengaturan penting yang bisa dimodifikasi:

Port 22 # Port default SSH PermitRootLogin no # Menonaktifkan login root PasswordAuthentication yes # Mengizinkan autentikasi password

Setelah mengubah konfigurasi, restart service SSH:

sudo systemctl restart ssh

6 Menghubungkan ke SSH Server

Dari komputer client, gunakan perintah:

ssh username@ip_address

Contoh:

ssh debian@192.168.1.100

7 Tips Keamanan

  • Gunakan key-based authentication daripada password
  • Ubah port default SSH
  • Batasi akses root
  • Gunakan fail2ban untuk mencegah brute force attacks
  • Regularly update sistem dan packages

Perintah Berguna

sudo systemctl start ssh # Memulai SSH service sudo systemctl stop ssh # Menghentikan SSH service sudo systemctl restart ssh # Restart SSH service sudo systemctl enable ssh # Enable SSH pada startup sudo systemctl disable ssh # Disable SSH pada startup
